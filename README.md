## 이상징후 효율적 탐지를 위한 IoT 메시지 로깅 시스템  

  
## 💻 프로젝트 소개
전동보장구와 센서에서 발생한 데이터 및 이벤트 메시지를 수집, 저장, 분석하여 발생한 이벤트를 시각화하여 실시간 모니터링이 가능한 시스템입니다.
<br>

## 📌 주요 기능
- 전동보장구로부터 들어오는 로그 데이터 로깅
- 이상징후를 감지하여 로깅
  - 양옆, 앞뒤 기울기에 따른 넘어짐 감지
  - 현재 배터리 사용량을 확인하여 에러 감지
  - GPS 에러 감지
  - 통신 에러 감지
